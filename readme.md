# informatiCup22

An attempt to solve the railway problem introduced in the [informatiCup2022](https://informaticup.github.io/competition/20-current) using simulated annealing.

## Usage

### Build

```shell
cargo run build
```

### Check

```shell
cargo check
```

### Run

```shell
cargo run "$(<path/to/input/file)"
```

## Documentation

```shell
cargo doc --open
```
